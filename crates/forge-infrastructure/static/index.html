Fluxo básico

          

            
1
Escolha uma IA
Ollama local se estiver rodando, ou configure uma chave Claude/Groq/OpenAI/Gemini.

            
2
Digite seu pedido
Use linguagem normal: “sou dentista e quero um site” ou “explique motor elétrico”.

            
3

Os badges mostram intenção, domínio, complexidade e técnica usada.

            
4
Envie para a IA
O Forge manda o prompt melhorado para o provider escolhido e mostra a resposta.

          

        

      


      

        

          
O que são providers?

          
Provider é a IA que responde. O Forge monta o prompt, mas quem gera a resposta final é Ollama, Claude, Groq, OpenAI, Gemini, OpenRouter, Mistral, DeepSeek, Together, Cerebras ou Hugging Face.

          

            
🟢
Verde
Está configurado e pode responder.

            
⚪
Cinza
Precisa de API key ou serviço local rodando.

            
FREE
Grátis ou com free tier
Ollama é local. Groq, OpenRouter, Cerebras e Hugging Face podem ter planos gratuitos ou modelos free, conforme regra de cada site.

            
↔
Troca rápida
Se uma IA der erro, abra Configurações, escolha outra IA configurada e envie novamente.

Como instalar e iniciar Ollama

          

            
1
Instalar
Baixe em ollama.com/download ou instale no Linux/WSL conforme a documentação oficial.

            
2
Iniciar serviço
Rode `ollama serve` se o serviço não iniciar sozinho.

            
3
Baixar modelo
Exemplo: `ollama pull qwen2.5-coder:7b` ou `ollama pull llama3.1:8b`.

            
4
Testar
Rode `curl http://localhost:11434/api/tags`. Se responder, o Forge detecta automaticamente.

          

          
Se o Forge disser que Ollama não está rodando, escolha outra IA em Configurações ou inicie o Ollama.
